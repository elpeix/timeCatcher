# Time catcher


```
cli usage: timecatchercli.py [-h] [-c] [-a] [-m MESSAGE] [-t] [-p] [-l] [-f FILE] [-s]

Catch time

optional arguments:
  -h, --help                    show this help message and exit
  -c, --catch                   Catch time
  -a, --add                     Add manual line to log
  -m MESSAGE, --message MESSAGE Message to log
  -t, --total                   Count total time
  -p, --print                   Print last entry
  -l, --log                     Print log
  -f FILE, --file FILE          File name
  -s, --sanitize                Sanitize log
```
