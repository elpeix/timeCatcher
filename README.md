# Time catcher



```
usage: timecatcher.py [-h] [-c] [-m MESSAGE] [-t] [-l] [-f FILE]

Catch time

options:
  -h, --help                    show this help message and exit
  -c, --catch                   Catch time
  -m MESSAGE, --message MESSAGE Message to log
  -t, --total                   Count total time
  -l, --log                     Print log
  -f FILE, --file FILE          File name
```
